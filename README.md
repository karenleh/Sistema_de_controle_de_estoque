# Sistema_de_controle_de_estoque
Sobre o Projeto

Este projeto consiste em um sistema de controle de estoque desenvolvido em Python, com foco em simplicidade e eficiência para gerenciamento de produtos. A aplicação permite cadastrar itens, controlar entradas e saídas e acompanhar o histórico completo de movimentações.

Os dados são armazenados em um banco de dados SQLite, garantindo persistência local, enquanto bibliotecas como Pandas e NumPy são utilizadas para processamento e análise das informações, como cálculo do valor total em estoque e média de preços.

Além disso, o sistema conta com um mecanismo de alerta para produtos com estoque abaixo do mínimo definido, ajudando na tomada de decisão. Também é possível exportar os dados para o formato JSON, facilitando integrações ou análises externas.

A aplicação é executada via terminal, sendo uma solução prática para estudos, pequenos projetos ou introdução ao desenvolvimento de sistemas com banco de dados.
